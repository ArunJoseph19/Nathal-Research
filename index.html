Nathal Manuscripts

Our research explores the potential of advanced AI techniques in deciphering the Nathal Manuscripts. A few examples of the manuscripts which span across different languages and scripts are shown here. We will extract the characters from such images for processing and correlating it to similar characters. More information on Nathal and the method can be found here.

Matching Manuscripts Characters

Here, for a set of pages which contain a similar character base, we extract the characters and process them through an Encoder Decoder model to form latent features. These latent features are used to cluster characters together, this gives us an insight as to the statistics of how many characters are similar and how often they occur. A few prominent cluster from a Japanese Type Script can be seen here.

Shape Context Implementation

Our other approach of relating characters to known languages, is to compare them with a database of Unicode characters which are saved as images using Noto fonts. We employ Shape Context similarity search to match the characters to their specifc unicode counterparts. These results give us an insight into the most recurrent languages, which inturn also helps us cluster them further to find similar characters in a larger database. More on Shape Context Algorithm can be found here.
